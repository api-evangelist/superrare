# SuperRare

SuperRare is a premier digital art marketplace and auction house built on Ethereum for discovering, collecting, and trading unique single-edition digital artworks as NFTs. Founded in 2017, the platform curates high-quality 1-of-1 pieces from vetted artists and operates as a decentralized autonomous organization (DAO) governed by RARE token holders.

## APIs

This repository catalogs the following SuperRare APIs and developer interfaces:

- **SuperRare Marketplace API** - REST API at `https://api.superrare.com` for artwork metadata, auction data, and Merkle proof flows
- **SuperRare Profile & Creator API** - OAuth-authenticated SDK/CLI for managing rare.xyz creator profiles, storefronts, products, posts, social, analytics, and events
- **Rare Protocol CLI** - Command-line tool for deploying ERC-721 collections, minting NFTs, managing auctions, and querying tokens across Ethereum Mainnet, Sepolia, Base, and Base Sepolia
- **Smart Contracts - The Bazaar** - Unified marketplace and auction house contract for reserve-price auctions and sale listings
- **Smart Contracts - Assets (ERC-721)** - SuperRareV1, SuperRareV2, Sovereign Series, and Space minting contracts
